# Medical Insurance EDA | Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Medical Insurance dataset using Python. The analysis focuses on identifying the factors that influence insurance charges and uncovering meaningful insights through statistical analysis and data visualization.

## Objectives

* Analyze the distribution of key variables
* Identify patterns affecting insurance charges
* Detect outliers and data quality issues
* Examine relationships between variables
* Generate actionable business insights

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Dataset Features

* Age
* Sex
* BMI
* Children
* Smoker
* Region
* Charges

## Analysis Performed

* Data Cleaning
* Missing Value Analysis
* Duplicate Record Analysis
* Univariate Analysis
* Outlier Detection
* Bivariate Analysis
* Correlation Analysis
* Business Insights Generation

## Key Insights

* Smokers incur significantly higher insurance charges than non-smokers.
* Insurance charges generally increase with age.
* BMI shows a weak positive relationship with insurance charges.
* Number of children has minimal impact on insurance charges.
* Male customers have slightly higher average insurance charges.
* Insurance charges are right-skewed with several high-value outliers.
